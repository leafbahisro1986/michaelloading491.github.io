What is Rufus? Rufus is a free utility that can be used to create bootable USB pen drives by burning ISO image to external drive. It is touted as one of the fastest applications around when creating a USB drive for a Windows or Linux installation. The tool can also be used for running low-level utilities, as well as for flashing firmware from DOS, and will also allow you to work on a machine that doesn't have an operating system installed. It is a light-weight tool that can run directly from the executable file, which means no installation is required.
