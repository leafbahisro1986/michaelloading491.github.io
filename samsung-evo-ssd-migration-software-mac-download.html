Cloned Drive Won’t Boot – How to Fix?

Some users find their Samsung SSD not booting after migration. Generally speaking, there areseveral common problems for a cloned disk fails to boot.

  • Cloned disk is not the first boot device. You’ll need to change boot order in theBIOS or UEFI Setup.

  • Wrong boot mode. If you clone a GPT disk, you’ll need to change Legacy to UEFI inBIOS as well, vice versa.

  • Start up error. You may need an installation disc to fix this problem. Boot from the disc, and select “Repair your computer” on the page of Install now. Then go“Troubleshoot” -> “Advanced Options” -> “Automatic Repair” to let it fix theproblem automatically.

  • Corrupted MBR or BCD. You’ll need to get into the Command Prompt with the help ofinstallation disc or repair disc. Then, type these commands successively and press Enterafter: bootrec.exe > bootrec.exe /fixmbr > bootrec.exe /fixboot >bootrec.exe /scanos > bootrec.exe /rebuildbcd. Finally, reboot yourcomputer.
